# Walle

Bilingual (EN/BG) AI-literacy site for Bulgarian high-schoolers, aged 15–18.

## Pages
- `index.html` — landing
- `prompting.html` — Lesson 01
- `spot-the-fakes.html` — Lesson 02
- `make-things.html` — Lesson 03
- `ai-for-school.html` — Lesson 04
- `fine-print.html` — The fine print

## Hosting
Static site — no build step. Served via GitHub Pages from the repo root.
CSS/JS are inlined into each page; `walle.css` / `walle.js` are the editable source of truth.
